# 逻辑与比较运算符

## 逻辑运算符

在python中,逻辑运算符有以下几种：

| 符号英文 |       作用        |
| :------: | :---------------: |
|   and    | 两真为真,一假为假 |
|    or    | 一真为真,两假为假 |
|   not    |     非真即假      |

## 运算符

基本运算符：

| 符号 | 作用 |  举例   | 符号 |   作用   |     举例      |
| :--: | :--: | :-----: | :--: | :------: | :-----------: |
|  +   | 相加 |  1+1=2  |  +=  | 自增操作 | i += 1(i=i+1) |
|  -   | 相减 |  2-1=1  |  -=  | 自减操作 | i -= 1(i=i-1) |
|  \*  | 相乘 | 2\*3=6  | \*\* |   乘方   |   2\*\*3=8    |
|  /   | 相除 | 3/2=1.5 |  //  |   整除   |    3//2=1     |
|  %   | 求余 |  3%2=1  | \*=  | 自乘操作 |    a \*= 2    |

<Note type="info"> 
python中的运算遵循四则运算顺序,可以用括号更改优先级
</Note>
在python中可以对字符串进行加乘操作：

```python
u = "python"
u = u*3
print(u) # "pythonpythonpython"
```

# 分支结构

## 单分支与双分支

```python
u = int(input("输入一个数字:"))
if u == 1:
    print("这个数等于1")
else:
    print("这个数不等于1")
```

<Note type="info"> 
这样的结构就是双分支,只有if就是单分支
</Note>
在python中,if的语法是这样的：

```python
if 条件: #如果真
    语句
else: #如果假
    语句
```

python依靠缩进来识别代码文件,所以代码必须使用缩进,建议使用以下几种格式：

- 一个空格(不推荐)
- 一个Tab
- 四个空格

尽量全部使用同一种进行缩进

## 多分支

### if elif型

```python
u = int(input("输入1-3的数字:"))
if u == 1:
    print("你输入了1")
elif u == 2:
    print("你输入了2")
elif u == 3:
    print("你输入了3")
else:
    print("未知数字")
```

在python中,多分支结构使用`elif`进行判断,使用规则同if 下面两幅图为双分支和多分支的区别：
![a.png](https://s2.loli.net/2022/01/12/sdjSE6kUNaKGLhP.png)
<Note type="info"> 
在一个多分支中只会执行一个代码块(最早的那一个)
所以有时会有`if...if...`这种写法,就可以执行多个代码块
</Note>

### match case型

**本条语法仅适用于python3.10以上,3.9及以下请用`elif多分支结构`**

最基本的字面量表示法:

```python
u = int(input("输入1-3的数字:"))
match u:
    case 1:
        print("你输入了1")
    case 2:
        print("你输入了2")
    case 3:
        print("你输入了3")
    case _:
        print("未知数字")
```

`case`代码块允许多个字面量以`|`分开,例如这样:

```python
case 301|302|307|308:
	print("redirect")
```
